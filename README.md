# Colllab Server - 实时协作文档服务

## 介绍
多人实时协作的文档服务器，文档编辑、实时同步、在线状态等

### 安装
\`\`\`bash
git clone https://github.com/Hahhhy/collab-server.git
cd TO
make setup
\`\`\`

### 运行
\`\`\`bash
make run
\`\`\`
